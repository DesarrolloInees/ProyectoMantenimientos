PROYECTO MANTENIMIENTOS:


Este proyecto se trata de una aplicación web, donde se va a poder tener todo el control sobre toda la operación de los Motorizados.
Desde el cargue de los servicios, pasando por la edición de los servicios, generación de reportes, administración y control de los inventarios,
administración y control de remisiones,  consulta de servicios realizados por parte de los tecnicos motorizados, además de la generación
de informes y generación de programación de servicios proximos.



## 🚀 Requisitos previos
Para ejecutar este proyecto, asegúrate de tener instalado:
- **PHP:** Versión [8.2.12]
- **Composer:** [(https://getcomposer.org/download/)]
- **Node.js/NPM:** [Para la gestión de dependencias de frontend](https://nodejs.org/en/download/current).
- **Servidor:** [XAMPP / Laragon / WAMP]
